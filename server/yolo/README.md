# YOLOv3 Model Files

The YOLOv3 model files are not included in the repository due to their size.

## Download the files

To download the required files, run the following commands:

```bash
curl -O https://pjreddie.com/media/files/yolov3.weights
curl -O https://raw.githubusercontent.com/pjreddie/darknet/master/cfg/yolov3.cfg
```

These commands will download:
- `yolov3.weights` - The pre-trained YOLOv3 model weights
- `yolov3.cfg` - The YOLOv3 model configuration file
